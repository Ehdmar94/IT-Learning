# Home Network Ethernet Installation with Switch

## Objective
Improve network stability and provide wired Ethernet connectivity to multiple devices.

## Project Summary
Installed Ethernet cabling from the living room to the secondary room through the crawlspace. Added a network switch to distribute the connection to multiple devices and created a permanent wall connection using a wall plate.

## Implementation
- Planned cable route through wall and crawlspace
- Installed Ethernet cable between rooms
- Installed wall plate for clean cable termination
- Connected Ethernet cable to network switch
- Connected multiple devices to wired network

## Outcome
- Established stable wired network connection
- Improved reliability and reduced latency compared to WiFi
- Network has remained operational for over one year

- Ethernet networking
- Network switching
