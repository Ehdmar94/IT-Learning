# Account Lockout Resolution

## Issue

User reported being unable to log into their workstation due to an account lockout.

## Troubleshooting

- Confirmed user was receiving an account lockout message
- Verified user account status in Active Directory
- Identified account was locked due to failed login attempts

## Resolution

- Unlocked user account
- Reset password
- Enabled password change at next login
- Guided user through login process

## Outcome

User successfully logged into workstation and regained access to company resources.
