# Slow Computer Performance

## Issue

User reported that their computer was running significantly slower than normal during daily tasks.

## Troubleshooting

- Confirmed user was experiencing slow system performance
- Guided user to open Task Manager
- Verified CPU usage and running processes
- Identified Google Chrome consuming excessive system resources due to multiple open tabs

## Resolution

- Instructed user to close unnecessary Chrome tabs and background processes
- Monitored system performance after changes were made

## Outcome

System performance improved significantly after reducing resource usage. User confirmed the computer was operating normally.
