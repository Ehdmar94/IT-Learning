# Black Screen Display Issue

## Issue

User reported laptop was powered on but the screen remained black.

## Troubleshooting

- Confirmed laptop was powered on
- Verified issue was not related to brightness settings
- Determined issue was not caused by sleep mode
- Identified potential display driver issue

## Resolution

- Instructed user to press Windows + Ctrl + Shift + B to reset the display driver
- Verified display functionality was restored

## Outcome

Screen returned to normal operation and user confirmed issue was resolved.
