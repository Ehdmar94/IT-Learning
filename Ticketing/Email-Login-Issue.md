# Email Login Issue (Cached Credentials)

## Issue

User reported being unable to access email and was repeatedly prompted to enter their password despite using the correct credentials.

## Troubleshooting

- Confirmed Caps Lock was disabled
- Reset user password to rule out credential issues
- Verified issue persisted after password reset
- Determined issue was related to locally stored credentials

## Resolution

- Guided user to remove and re-add email account within the email application
- Refreshed stored account credentials

## Outcome

User successfully accessed email after account reconfiguration.
