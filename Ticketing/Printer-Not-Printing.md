# Printer Not Printing

## Issue

User reported that a printer was powered on but would not print documents.

## Troubleshooting

- Confirmed printer was connected to WiFi
- Verified printer had paper loaded and displayed no error messages
- Guided user to review printer settings on their computer
- Identified multiple printer instances listed in Windows
- Verified the physical printer name

## Resolution

- Instructed user to select the correct printer instead of a duplicate printer entry
- Tested print functionality using the correct printer

## Outcome

Printer successfully processed print jobs. User confirmed printing functionality was restored.
