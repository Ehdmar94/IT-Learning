# Power BI Access Issue (Account Mismatch)

## Issue
User was unable to access customer data within Microsoft Power BI.

## Troubleshooting

- Confirmed user could access Power BI
- Verified application was functioning normally
- Determined user was logged into a personal Power BI account
- Identified account mismatch preventing access to customer data

## Resolution

- Instructed user to sign out of personal Power BI account
- Guided user to sign in using the customer-associated email account

## Outcome

User successfully accessed customer data after signing into the correct account.
